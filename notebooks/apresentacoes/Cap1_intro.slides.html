Descrição do Problema¶

Uma das causas desse problema que h uma altssima barreira de entrada para a pesquisa aplicada nessa rea, so elas:

  1. A primeira barreira seria dominar todas as reas relacionadas e necessrias para elaborar uma estratgia de investimentos.
  2. Restries oramentrias limitam o tamanho de uma equipe. Isso um problema, porque uma equipe enxuta de pesquisadores tem amplitude limitada.
  3. Direitos de confidencialidade do provedor dados impede que esses dados sejam disponibilidados publicamente.
  4. Incentivo a falsos positivos. Pesquisadores so remunerados para produzir simulaes de estratgias de investimentos com resultados promissores em um conjunto de dados limitado, levando a um vis de seleo (overfitting).

Fonte: DE PRADO, M. L., FABOZZI, F. J. Crowdsourced Investment Research Through Tournaments, The Journal of Financial Data Science, v. 2, n. 1, pp. 8693, 2020.

Uma Alternativa para o problema¶

De Prado (2020), sugere a formao de um torneio, onde um organizador formaliza um desafio de estratgias investimento em termos de um problema preditivo, para incentivar a pesquisa de maneira coletiva.

Os torneios dividem o problema de estratgias investimento em duas tarefas especializadas:

  • i. Estruturar os dados de forma a constituir um problema preditivo, que executada pelos organizadores do torneio, que possuem extenso conhecimento em finanas;

  • ii. Modelagem, que feita pelos participantes do torneio com conhecimento de cincia de dados.

Como Suplantar as barreiras através do torneio¶

  1. Como o problema de previso j foi definido os participantes do torneio podem trabalhar nesse desafio mesmo que no possuam conhecimento em finanas.
  2. O torneio acessvel a ampla comunidade de cientistas de dados.
  3. A confidencialidade resolvida ofuscando ou encriptando os dados de forma a preservar as propriedades matemticas essencial para um problema preditivo.
  4. O overfitting mitigado remunerando os pesquisadores atravs do desempenho fora da amostra. Alm disso os pesquisados so incentivados a apostar o prprio dinheiro em suas previses.

Fonte: DE PRADO, M. L., FABOZZI, F. J. Crowdsourced Investment Research Through Tournaments, The Journal of Financial Data Science, v. 2, n. 1, pp. 8693, 2020.

Esse Torneio já existe¶

Modelos de torneio existentes¶

  1. Numerai Signals: Os competidores so livres para usar qualquer tipo de dados que dispuserem referentes a uma cesta de 5000 aes divulgadas de forma explcita. A partir dos prprios dados devem treinar um modelo preditivo capaz de prever o comportamento de cada dessas aes no futuro.
  2. Numerai Tournament: Essa a competio principal, configurada exatamente da forma descrita por De Prado (2020), usando os dados encriptados disponibilizados pela equipe da numerai e as predies so feitas nos mesmas aes da Numerai Signals, porm anonimizados.

Adicionando Indicadores ao modelo¶

Agora que j temos os preos, precisamos configurar indicadores que adicionem valor preditivo ao modelo.

Índice de Força Relativa (RSI):¶

O RSI um indicadorde anlise tcnica que busca medir se o ativo est excessivamente vendido (RSI>70) ou excessivamente comprado (RSI<30).

$$RSI = 100 - (100/(1 + RS))$$

RS = Mdia dos retornos nos perodos de alta / Mdia dos retornos nos perodos de baixa

Fonte: WILDER, J.New Concepts in Technical Trading Systems. Trend Research, 1978

Modelos de Machine learning funcionam bem com dados distribudos na forma de histograma. Aqui faremos uma diviso em 5 bins.

A Variável Alvo (Target)¶

  • A numerai responsvel por disponibilizar a varivel alvo para cada ativo, para a prxima era. Uma era comea toda sexta-feira e ser baseada nos retornos dos dias subsequentes.

  • A varivel alvo configurada como um problema de classificao, normalizadas e neutralizadas dos retornos dentro de cada era.
